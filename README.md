# Muhammad Hamza - Portfolio Website

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Next.js](https://img.shields.io/badge/Next.js-14.x-black)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3.x-38B2AC)
![Status](https://img.shields.io/badge/status-active-success.svg)

A modern, responsive personal portfolio website showcasing professional experience, projects, and skills. Built with Next.js and styled with Tailwind CSS for optimal performance and elegant design.

## 🌟 Features

- **Responsive Design** - Fully responsive layout that works seamlessly across all devices
- **Modern UI/UX** - Clean, professional interface with smooth animations and transitions
- **SEO Optimized** - Built-in meta tags and structured data for better search engine visibility
- **Fast Performance** - Optimized with Next.js static generation for lightning-fast load times
- **Interactive Navigation** - Smooth scroll navigation with active section highlighting
- **Accessible** - WCAG compliant with proper ARIA labels and semantic HTML
- **PWA Ready** - Progressive Web App support with manifest and service worker
- **Dark Theme** - Professional dark color scheme with teal accents

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) - React framework for production
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- **Language:** JavaScript/HTML
- **Fonts:** Custom web fonts (WOFF2 format)
- **Icons:** Custom favicon set for all platforms
- **Analytics:** Google Analytics integration

## 📁 Project Structure

```
├── css/                    # Compiled CSS stylesheets
├── fonts/                  # Web fonts (WOFF2 format)
├── images/                 # Favicons and images
├── js/                     # JavaScript bundles
├── index.html              # Main HTML file
├── site.webmanifest        # PWA manifest
└── README.md               # Project documentation
```

## 🚀 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hamzaka430/hamzaka.git
   cd hamzaka
   ```

2. **Serve the website**
   
   Using Python:
   ```bash
   python -m http.server 8000
   ```
   
   Or using Node.js:
   ```bash
   npx http-server
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```

## 💻 Usage

This is a static portfolio website that can be:

- **Deployed to any static hosting service** (GitHub Pages, Netlify, Vercel, etc.)
- **Served from any web server** (Apache, Nginx, etc.)
- **Opened directly in browser** (for local development)

### Customization

To customize the portfolio for your own use:

1. Update personal information in `index.html`
2. Replace images in the `images/` folder
3. Modify styles in `css/` directory
4. Update meta tags and SEO information
